# `settings.json`

本页介绍`settings.json`的具体结构与作用。

```json filename="settings.json" copy
{
    "images": {
        "avatar": "",
        "logo_big": "",
        "logo_small": "",
        "language_flags": {}
    },
    "others": {
        "colors": [],
        "defaultLanguage": "",
        "serverHost": "",
        "serverPort": ""
    },
    "theme": {
        "primaryColor": "",
        "primaryColorHover": ""
    }
}
```

## `images`

用于设置各种图片。

### `images.avatar`

用于指定回答问题时显示的头像，一般是网站主人的头像。

类型：`String`

默认值：`./assets/Avatar.jpg`

![DefaultAvatar](../images/Avatar.jpg)

### `images.logo_big`

位于网站左上角的Logo。

类型：`String`

默认值：`./assets/logo.png`

### `images.logo_small`

移动端位于网站左上角的Logo。

类型：`String`

默认值：`./assets/logoMobile.png`

### `language_flags`

语言旗帜图标。

类型：`Object`

默认值：

``` json
{
    "enus": "./assets/flags/gb.png",
    "zhcn": "./assets/flags/cn.png",
    "zhtw": "./assets/flags/tw.png",
    "ja": "./assets/flags/jp.png",
    "de": "./assets/flags/de.png",
    "zhsb": "./assets/flags/unknown.png"
}
```

这些图标都是从[flagcdn](https://flagcdn.com/)下载来的，flagcdn在一些地区无法访问，所以都存在了仓库里，你可以给改成任何链接。

例子：

``` json copy
{
    "enus": "https://flagcdn.com/40x30/gb.png"
}
```

如果您在中国大陆，强烈建议加入下面这一条将以确保您的网站能够正常上线。

``` json copy
{
    "zhtw": "https://flagcdn.com/40x30/hk.png"
}
```

## `others`

其他设置。

### `others.colors`

提问者头像背景随机颜色列表，请使用十六进制颜色。

类型：`Array`

成员类型：`String`

默认值：`[ "#c3cfe2" ]`

举例：`[ "#c3cfe2", "#8a2be2"]`

### `others.defaultLanguage`

默认显示语言，第一次访问网站时显示此语言，之后跟随Cookie。

类型：`String`

默认值：`enus`

可用语言：

| `defaultLanguage`           | Language Name In Website | Language    |
| --------------------------- | ------------------------ | ----------- |
| `enus`                      | English                  | 英语        |
| `zhcn`                      | 简体中文                  | 简体中文    |
| `zhtw`                      | 繁體中文                  | 繁体中文    |
| `ja`                        | 日本語                    | 日语        |
| `de`                        | Deutsch                  | 德语        |
| `ru`                        | Русский                  | 俄语        |
| `zhsb`                      | 减啼忠吻                  | ？？？      |

### `others.serverHost`

服务端地址。

类型：`String`

默认值：`127.0.0.1`

举例：`https://example.com`

### `others.serverPort`

服务端端口。

类型：`String`

默认值：`1106`

## `theme`

主题色等。

### `theme.primaryColor`

网站主题色，请使用十六进制颜色。

类型：`String`

默认值：`#8a2be2`

### `theme.primaryColorHover`

网站鼠标悬停主题色，请使用十六进制颜色。

类型：`String`

默认值：`#8a2be2`
