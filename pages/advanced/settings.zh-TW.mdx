# `settings.json`

本頁介紹`settings.json`的具體結構與作用。

```json filename="settings.json" copy
{
    "images": {
        "avatar": "",
        "logo_big": "",
        "logo_small": "",
        "language_flags": {}
    },
    "others": {
        "colors": [],
        "defaultLanguage": "",
        "serverHost": "",
        "serverPort": ""
    },
    "theme": {
        "primaryColor": "",
        "primaryColorHover": ""
    }
}
```

## `images`

用於設定各種圖片。

### `images.avatar`

用於指定回答問題時顯示的頭像，一般是網站主人的頭像。

類型：`String`

預設值：`./assets/Avatar.jpg`

![DefaultAvatar](../images/Avatar.jpg)

### `images.logo_big`

位於網站左上角的Logo。

類型：`String`

預設值：`./assets/logo.png`

### `images.logo_small`

移動端位於網站左上角的Logo。

類型：`String`

預設值：`./assets/logoMobile.png`

### `language_flags`

語言旗幟圖示。

型別：`Object`

預設值：

``` json
{
    "enus": "./assets/flags/gb.png",
    "zhcn": "./assets/flags/cn.png",
    "zhtw": "./assets/flags/tw.png",
    "ja": "./assets/flags/jp.png",
    "de": "./assets/flags/de.png",
    "zhsb": "./assets/flags/unknown.png"
}
```

這些圖示都是從[flagcdn](https://flagcdn.com/)下載來的，flagcdn在一些地區無法訪問，所以都存在了倉庫裡，你可以給改成任何連結。

例子：

``` json
{
    "enus": "https://flagcdn.com/40x30/gb.png"
}
```

## `others`

其他設定。

### `others.colors`

提問者頭像背景隨機顏色列錶，請使用十六進制顏色。

類型：`Array`

成員類型：`String`

預設值：`[ "#c3cfe2" ]`

舉例：`[ "#c3cfe2", "#8a2be2"]`

### `others.defaultLanguage`

預設顯示語言，第一次訪問網站時顯示此語言，之後跟隨Cookie。

類型：`String`

預設值：`enus`

可用語言：

| `defaultLanguage`           | Language Name In Website | Language    |
| --------------------------- | ------------------------ | ----------- |
| `enus`                      | English                  | 英語        |
| `zhcn`                      | 简体中文                  | 簡體中文    |
| `zhtw`                      | 繁體中文                  | 繁體中文    |
| `jajp`                      | 日本語                    | 日語        |
| `de`                        | Deutsch                  | 德語        |
| `ru`                        | Русский                  | 俄語        |
| `zhsb`                      | 減啼忠吻                  | ？？？      |

### `others.serverHost`

服務端地址。

型別：`String`

預設值：`127.0.0.1`

举例：`https://example.com`

### `others.serverPort`

服務端連接埠。

型別：`String`

預設值：`1106`

## `theme`

主題色等。

### `theme.primaryColor`

網站主題色，請使用十六進制顏色。

類型：`String`

預設值：`#8a2be2`

### `theme.primaryColorHover`

網站滑鼠懸停主題色，請使用十六進制顏色。

類型：`String`

預設值：`#8a2be2`
