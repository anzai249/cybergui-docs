# `secret.json`

本页介绍`secret.json`的具体结构与作用。

注意，本json没有默认值，请将除`mysql.dev`以外所有配置项全部配置好。

```json filename="secret.json" copy
{
  "mysql": {
    "dev": {
      "username": "",
      "password": "",
      "name": "",
      "host": ""
    },
    "prod": {
      "username": "root",
      "password": "root",
      "name": "cybergui",
      "host": "127.0.0.1"
    }
  },
  "password": "admin"
}
```

## `mysql`

用于设置MySQL数据库。

### `mysql.dev`

此为开发时debug使用的配置项，请尽量不要设置。

### `mysql.prod`

生产环境使用的MySQL设置。

#### `mysql.prod.username`

MySQL用户名。

类型：`String`

#### `mysql.prod.password`

MySQL用户密码。

类型：`String`

#### `mysql.prod.name`

MySQL数据库库名。

类型：`String`

#### `mysql.prod.host`

MySQL数据库地址。

类型：`String`

## `password`

回答问题时使用的验证密码。

类型：`String`
