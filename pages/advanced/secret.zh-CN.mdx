# `secret.json`

本页介绍`secret.json`的具体结构与作用。

<Callout type="warning" emoji="⚠️">
  注意，本json没有默认值，请将除`mysql.dev`以外所有配置项全部配置好。
</Callout>

```json filename="secret.json" copy
{
  "mysql": {
    "dev": {
      "username": "",
      "password": "",
      "name": "",
      "host": ""
    },
    "prod": {
      "username": "root",
      "password": "root",
      "name": "cybergui",
      "host": "127.0.0.1"
    }
  },
  "password": "admin",
  "https": true,
  "serverPort": 1106,
  "key": "path/to/your.key",
  "cert": "path/to/your.pem",
  "review": false
}
```

## `mysql`

用于设置MySQL数据库。

### `mysql.dev`

此为开发时debug使用的配置项，请尽量不要设置。

### `mysql.prod`

生产环境使用的MySQL设置。

#### `mysql.prod.username`

MySQL用户名。

类型：`String`

#### `mysql.prod.password`

MySQL用户密码。

类型：`String`

#### `mysql.prod.name`

MySQL数据库库名。

类型：`String`

#### `mysql.prod.host`

MySQL数据库地址。

类型：`String`

## `password`

回答问题时使用的验证密码。

类型：`String`

## `https`

服务端是否使用https，建议打开。

类型：`Bool`

## `serverPort`

服务端开放端口。

类型：`Number`

## `key`

SSL证书.key文件路径

类型：`String`

## `cert`

SSL证书.pem文件路径

类型：`String`

## `review`

问题是否需要审核之后再展示。
 
<Callout type="warning" emoji="⚠️">
  请把`settings.json`中的`others.review`设置的和这个一样以免出现怪异的bug。
</Callout>

类型：`Boolean`
