# `secret.json`

本頁介紹`secret.json`的具體結構與作用。

<Callout type="warning" emoji="⚠️">
  注意，本json沒有預設值，請將除`mysql.dev`以外所有配置項全部配置好。
</Callout>

```json filename="secret.json" copy
{
  "mysql": {
    "dev": {
      "username": "",
      "password": "",
      "name": "",
      "host": ""
    },
    "prod": {
      "username": "root",
      "password": "root",
      "name": "cybergui",
      "host": "127.0.0.1"
    }
  },
  "password": "admin",
  "https": true,
  "serverPort": 1106,
  "key": "path/to/your.key",
  "cert": "path/to/your.pem",
  "review": false
}
```

## `mysql`

用於設定MySQL資料庫。

### `mysql.dev`

此為開發時debug使用的配置項，請儘量不要設定。

### `mysql.prod`

生產環境使用的MySQL設定。

#### `mysql.prod.username`

MySQL使用者名稱。

型別：`String`

#### `mysql.prod.password`

MySQL使用者密碼。

型別：`String`

#### `mysql.prod.name`

MySQL資料庫庫名。

型別：`String`

#### `mysql.prod.host`

MySQL資料庫地址。

型別：`String`

## `password`

回答問題時使用的驗證密碼。

型別：`String`

## `https`

服務端是否使用https，建議開啟。

型別：`Bool`

## `serverPort`

服務端開放埠。

型別：`Number`

## `key`

SSL證書.key檔案路徑

型別：`String`

## `cert`

SSL證書.pem檔案路徑

型別：`String`

## `review`

問題是否需要稽核之後再展示。
 
<Callout type="warning" emoji="⚠️">
  請把`settings.json`中的`others.review`設定的和這個一樣以免出現怪異的bug。
</Callout>

型別：`Boolean`
