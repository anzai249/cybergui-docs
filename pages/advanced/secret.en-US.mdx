# `secret.json`

This page will introduce detailed usage of `secret.json`.

Note: This json has no defaults, so please configure all configuration items except `mysql.dev`.

```json filename="secret.json" copy
{
  "mysql": {
    "dev": {
      "username": "",
      "password": "",
      "name": "",
      "host": ""
    },
    "prod": {
      "username": "root",
      "password": "root",
      "name": "cybergui",
      "host": "127.0.0.1"
    }
  },
  "password": "admin"
}
```

## `mysql`

Used to set up the MySQL database.

### `mysql.dev`

Used for debugging during development, so please try not to set it.

### `mysql.prod`

MySQL settings used by the production environment.

#### `mysql.prod.username`

MySQL username.

Type: `String`

#### `mysql.prod.password`

MySQL user password.

Type: `String`

#### `mysql.prod.name`

MySQL database name.

Type: `String`

#### `mysql.prod.host`

MySQL host.

Type: `String`

## `password`

Password used to answering questions.

Type: `String`
